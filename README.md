# RasPi-KY-026-Flame-Sensor
This uses a Raspberry Pi 4 and a KY-026 to detect a flame.
Pin Wiring

KY-026   |   Raspberry Pi  |   Buzzer
Positive |   3.3v/Pin 1    |  
Negative |   Ground/Pin 9  | 
Signal   |   GPIO 17/Pin 11|   
L        |   Not Attached  |
         |   GPIO 27/Pin 13|   Positive
         |   Ground/Pin 6  |   Negative
